# Monacle Cloud

Secure cloud storage for Monad.

## Features

- **Monad ID Login**: Only accessible to Monad members.
- **Personal & Team Spaces**: Separate storage areas.
- **File Management**: Upload, Delete, Share.
- **Modern UI**: Aurora Mesh design system.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000).

## Configuration

- **Database**: Configured to connect to `https://monadb.snowman0919.site`.
- **Auth**: Monad ID integration. Set `MONAD_CLIENT_ID` environment variable if needed.
